# 📊 Retail Sales Analysis Dashboard – Tableau

## 📌 Project Overview
This project is an interactive Retail Sales Analysis Dashboard built using Tableau to analyze sales performance, profitability, customer behavior, and regional trends.

The dashboard provides business insights through KPIs, dynamic filtering, and advanced visualizations and Navigation to link between The Dashboards .

---

## 🎯 Business Objectives
- Monitor total sales and profit performance
- Identify top and bottom performing products
- Analyze customer distribution and behavior
- Track monthly profit trends
- Compare regional sales performance

---

# 📍 Dashboard 1: Retail Sales Analysis (Overview)

##  KPIs
- Total Sales
- Quantity by Category
- Top N Products with Profit
- Profit Over Months
- Profit with Quantity by State (Map)
- Average Sales Price per Region
# Used Parameters and Calculated Fields to Control Which KPI To Display
## 🎛 Parameters Used

### 🔹 Top N Parameter
- Data Type: Integer
- Allowable Values: Range
- Used to dynamically control ranking of:
  - Top Products
  - Top Customers
  - Bottom Products

---

## 🎛 Filters
- Category Filter
- Month Filter
- Gender Filter
- KPI Selector
- Show/Hide Products Toggle

---

## 🎬 Dashboard Actions
- Filter Actions (click to filter visuals)
- Navigation Buttons (Overview ↔ Details)
